# Manual Regression Checklist

Use this checklist before and after larger Phase 2 refactors.

- [ ] Demo entry works
- [ ] Visitor code works
- [ ] Admin login works
- [ ] Admin code generation works
- [ ] Map pins load
- [ ] Recommended filter works
- [ ] Quiz stamp updates passport
- [ ] Chat responds
- [ ] Chat limit blocks correctly
- [ ] Chat site-specific question: "What should I know about Bangunan Sultan Abdul Samad?"
- [ ] Chat general trail question: "Which must-visit sites should I start with on this walk?"
- [ ] Chat unsupported question: "Can you recommend stock investments for this week?"
- [ ] Demo reset clears only demo data
- [ ] Logout returns to landing page

Suggested baseline screen capture flows:

- Explore Demo
- Visitor Passkey validation
- Project Admin login
- Admin passkey generation
- AI chat quota
- Reset Demo Progress
- Passport and quiz completion
